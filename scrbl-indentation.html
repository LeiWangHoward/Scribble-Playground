1. Scribble Indentation

This document has two sections: scribble indentation rules and scribble indentation functions.

1 1.1 Scribble Indentation Rules

The scribble indentation rules are created based on the judgement of parent parenthesis. It will perform the indentation on each line of code based on two facts: check parent parenthesis and counting number of parenthesis till the outmost @ notation that the current line is in.

2 1.1.1 Parenthesis judgement rules

  • Tab adjusted:

    @itemlist[@item{item1}

              @item{item2}

  • One-space adjusted:

    @centered{

     @bold{blah}

     @italic{blah}

     }

    @itemlist[

     @item{blah}

     @item{blah}]

Line breaking rules:

1)Only handle string inside {} or pure string

2)Each paragraph shall not have more/less than given character number(e.g. 70). The last few "words" will be passed to the next line, or some "words" from the next line will be pass back to current line
