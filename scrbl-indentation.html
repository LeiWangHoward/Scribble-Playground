1. Scribble Indentation

This document has two sections: scribble indentation rules and scribble indentation functions.

1 Scribble Indentation Rules

The scribble indentation rules are created based on the judgement of parent parenthesis. It will perform the indentation on each line of code based on two facts: check parent parenthesis and counting number of parenthesis till the outmost @ notation that the current line is in. Then the program will put spaces before each line of code based on how many parent parenthesis outside and the following parenthesis judgement rules

  • Tab adjusted: exists items at the same line after [

    @itemlist[@item{item1}

              @item{item2}]

  • One-space adjusted:

    @centered{

     @bold{blah}

     @italic{blah}

     }

    @itemlist[

     @item{blah}

     @item{blah}]

2 Line breaking rules

  • Only handle text or text inside { }

  • Each line of code’s length cannot exceed the given paragraph width limit(e.g. 70 characters per line). When exceed the limitation, the last few words seperated by #\space will be passed to the next line of code
